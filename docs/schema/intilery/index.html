Intilery Default Schema

The following events are reserved events in the Intilery Default Schema, you should not use these Event names/types unless for their intended purpose.

These Events can be founf in the Intilery app on the Schema page shown below

Default Events#

Mobile#

Application Lifecycle Events#

  • Application Installed
  • Application Opened
  • Application Updated
  • Application Backgrounded
  • Application Crashed
  • Application Uninstalled

Campaign Events#

  • Push Notification Received
  • Push Notification Tapped
  • Push Notification Bounced
  • Install Attributed
  • Deep Link Clicked
  • Deep Link Opened
  • Install Attributed

Email#

  • Email Bounced
  • Email Delivered
  • Email Link Clicked
  • Email Marked as Spam
  • Email Opened
  • Unsubscribed

API#

  • Identify
  • Track
  • Page
  • Screen

Industry#

  • Retail/Ecommerce
  • Gaming
  • Travel
  • Insurance
